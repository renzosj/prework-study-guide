HTML

  • The head element contains information about the webpage.
  • The information stored in the head element are called metadata.
  • Metadata are invisible to users, but contain information regarding search engines(SEO) accesibility, and performance.
  • The only visible element in the head is the title which appears in browser tabs.

  • The body element represents the visible conent shown to the user.
  • Many elements go into the body such as the main section, headers, paragraphs, images, links, lists, and so on.
  • The main section contains the core contents of a webpage.
  • It is a parent element that is used to organized other body related elements.
  • Header/heading elements are useful subtitles that can be used to organize content within the webpage itself.
  • You can have up to six tiers of headers, but it's best practice to use the first header only once.
  • Paragraph elements can hold blocks of text.
  • List elements can be ordered or unordered.
  • Some elements like the img element has required attributes that need to be specified in order to function properly.

  • Elements follow a child-parent terminology, with which nested elements are the children of larger scope elements.
  • This forms an organizational heirarchy such as list items being nested in an ordered or unordered list, which is nested in a section element, in turn nested in a main element, in turn nested in a body element.

CSS

  • CSS stands for Cascading Style Sheets
  • There are three methods to styling a webpade using CSS: inline CSS, internal CSS style sheet, and external CSS style sheet
  • The CSS Rule refers to two main parts of a piece of CSS code: the selector and declaration block
  • The selector defines the element or attributes to which the stylings will apply
  • The declarations are made up of property and value pairings aka the styling rules that are being applied to elements
  • The CSS box model refers to the box like nature of HTML elements, adjustable via margins and paddings
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • These are common commands related to the GitHub workflow
  • git status: check what branch we are currently on
  • git checkout [branch name]: switch to specified branch
  • git checkout -b [branch name]: create new branch with specified name and switch to it
  • git add -A: stages all files in current branch for a commit
  • git commit -m: confirm commits to the branch + message that describes the change
  • git pull origin main: pulls in main branch from remote to local storage
  • git push origin [branch name]: records local feature branch + commits over to remote storage

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • The Javascript control flow goes from top to bottom
  • A function is a customizable set of instructions that can be reused over and over
  • A conditional follows if-then logic, and is a way to execute code only if certain conditions are met
